# LuxeLane — Luxury E-Commerce Storefront

A fully functional e-commerce website built from scratch using HTML, CSS, and JavaScript. No frameworks, no templates.

🔗 **Live Site:** [luxelane1593.netlify.app](https://luxelane1593.netlify.app)

## Features

- 🛒 Add to Cart system with quantity tracking and total calculation
- ❤️ Wishlist toggle on every product
- 🔍 Real-time product search and filter
- 📦 Offer modal with promo code copy feature
- 🔔 Toast notifications for all user actions
- 📱 Mobile responsive with hamburger menu
- ✨ Scroll reveal animations
- 🔝 Back to top button
- 🎨 Navbar scroll shadow effect

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome (icons)
